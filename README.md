# Tbx1-snRNAseq-pipeline

Questo repository GitHub contiene la pipeline di analisi, gli script e i materiali relativi al lavoro di tesi magistrale di **Giacomo Pensiero**.

## Informazioni accademiche
- **Università:** Università degli Studi di Napoli Federico II  
- **Corso di Laurea Magistrale:** Biotecnologie del Farmaco  
- **Ambito della tesi:** Bioinformatica  

## Titolo della tesi
**Alterazioni trascrizionali nell’orecchio interno nel modello Tbx1+/- della sindrome 22q11.2: ri-analisi snRNA-seq con approcci di co-espressione genica**

## Descrizione del progetto
Questo progetto riguarda la ri-analisi di dati di single-nucleus RNA sequencing (snRNA-seq) per studiare le alterazioni trascrizionali associate all’aploinsufficienza di **Tbx1** in tessuti dell’orecchio interno murino.
